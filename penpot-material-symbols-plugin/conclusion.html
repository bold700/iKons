Waarom localhost problematisch is:

  • CORS Preflight: Penpot stuurt OPTIONS requests die Python server niet ondersteunt

  • Browser Security: Cross-origin restricties

Je had gelijk: Localhost plugins zouden moeten werken, maar er zijn technische beperkingen in de huidige setup.
