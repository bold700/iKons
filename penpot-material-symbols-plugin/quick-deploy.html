🚀 Snelle Plugin Deployment

⚠️ GitHub Pages duurt te lang?

Gebruik Netlify voor onmiddellijke deployment!

📦 Stap 1: Download Plugin ZIP

De plugin bestanden zijn al klaar in een ZIP bestand.

Locatie: material-symbols-plugin.zip

🌐 Stap 2: Netlify Drop

  1. Ga naar netlify.com/drop
  2. Sleep material-symbols-plugin.zip op de pagina
  3. Wacht 30 seconden
  4. Krijg een URL zoals: https://random-name.netlify.app

✅ Stap 3: Gebruik in Penpot

Gebruik de Netlify URL in Penpot:

https://random-name.netlify.app/manifest.json

🔄 Alternatief: Wacht op GitHub Pages

Als je liever GitHub gebruikt, controleer over 5-10 minuten:

https://bold700.github.io/iKons/manifest.json
