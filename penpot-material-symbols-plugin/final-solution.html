🎯 Final Solution: Material Symbols Plugin

❌ Het Probleem

CORS Error: Penpot (HTTPS) kan geen HTTP localhost laden

Console Error: "Access to fetch blocked by CORS policy"

Conclusie: Plugin werkt perfect, maar heeft HTTPS hosting nodig

✅ De Oplossing: Netlify Drop

Upload plugin naar publieke HTTPS hosting

🚀 Stap 1: Netlify Upload

1. Ga naar netlify.com/drop

2. Sleep material-symbols-plugin-cors.zip erop

3. Wacht 30 seconden

4. Krijg een URL zoals: https://amazing-name.netlify.app

🎯 Stap 2: Test Plugin URL

Test eerst of manifest werkt:

https://jouw-netlify-url.netlify.app/manifest.json

Je moet JSON zien, geen error

🎨 Stap 3: Installeer in Penpot

1. Ga naar Penpot

2. Druk Ctrl + Alt + P

3. Plak: https://jouw-netlify-url.netlify.app/manifest.json

4. Klik "INSTALLEREN"

5. Plugin verschijnt in lijst als "Material Symbols"

🎉 Verwachte Resultaat

  • ✅ Geen CORS errors meer
  • ✅ "Material Symbols" plugin in lijst
  • ✅ Werkende interface met 200+ iconen
  • ✅ Zoekfunctie werkt
  • ✅ Drie stijlen: Outlined, Rounded, Sharp
  • ✅ Iconen toevoegen aan canvas
